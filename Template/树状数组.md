# 树状数组

## 1. 模板

```c++
    int n;
    vector<int> tree;

    int lowbit(int x) {
        return x & -x;
    }
    
    int query(int x) {
        int ans = 0;
        for(int i = x; i > 0; i -= lowbit(i))
            ans += tree[i];
        return ans;
    }

    void update(int x, int u) {
        for(int i = x; i <= n; i += lowbit(i))
            tree[i] += u;
    }
```

## 2. 初始化

```c++
    // BIT从1开始, 需要整体往右偏移
    n = nums.size();
    tree.resize(n + 1, 0);
```
